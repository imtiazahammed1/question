JavaScript is called a Client-side Scripting Language and one of the most commonly used scripting languages (1).That means it is a computer programming language that runs inside the web-browsers but one important thing to remember is that client's web-browser also needs to support the JavaScript Code (2). That's why now-a-days most of the modern web browsers have their JavaScript engines (3). For example, Google Chrome has its own JavaScript engine called V8 and Mozilla Firefox has its own JavaScript engine called SpiderMonkey (4).

So, whenever we run a JavaScript program inside a web browser, JavaScript code is received by the browser's engine and the engine will execute each line of the file from top to bottom (5).

First of all, JavaScript engine will parse the code line by line and and if an error is detected by the parser, it throws a kind of error and stops execution of the code and if there are no mistakes/errors in the code, it creates the data structure called AST (it stands for Abstract Syntax Tree) (6).

Once the Abstract Syntax Tree is created by the parser, the JavaScript engine converts the JavaScript code into the machine code and then execute it (7).

And this is why JavaScript actually works (8).

The difference between local storage and session storage is given below:

i) How to store: LocalStorage stores data with no expiration date whereas SessionStorage stores data for one session that means data is lost when the browser tab is closed.

ii) Better Choice: SessionStorage is a better choice over LocalStorage because session data is cleared when the browser tab is closed.

iii) How to delete: LocalStorage data need to be deleted via JavaScript by using clear method of local storage [For ex: localStorage.clear()] whereas SessionStorage data is deleted when the browser tab or page session ends.

iv) Accessible from: We can access the LocalStorage from any window whereas SessionStorage needs the same tab for access.

v) Capacity: LocalStorage capacity is 5-10 Mbs(Browser Dependent) whereas SessionStorage capacity is always about 5 Mb.
